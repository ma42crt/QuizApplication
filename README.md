Project Title . Holton College Digital Quiz System
Profect Discription  
this project is a console based digital quiz application developed using pythone. this system allow user to login, answer the questions's options and get the final score and percentage at the end of quiz
Features

User login system

Multiple-choice quiz questions

Immediate feedback for correct and incorrect answers

Final score and percentage calculation

Save results to a text file

Option to replay the quiz

How to Run the Program

Make sure Python 3 is installed on your computer.

Download or clone this repository.

Open a terminal/command prompt in the project folder.

Run the following command:

python quiz.py
